## 🎯 메인페이지 핵심 목표

### 비즈니스 목표
1. **5초 안에 가치 전달**: "블로그 자동 발행 솔루션"임을 즉시 이해
2. **무료 체험 전환**: 방문자 → 회원가입 전환율 30% 목표
3. **신뢰 구축**: 실제 작동하는 자동화 시스템임을 증명
4. **SEO 최적화**: "블로그 자동 발행", "AI 블로그 작성" 키워드 노출

### 사용자 목표

- **쉬운 시작**: 클릭 몇 번으로 무료 체험 시작
- **안심**: 가격, 기능, 후기를 통해 신뢰 확보

### 콘텐츠

**문제 (Pain Points)**
1. ❌ **트렌드 추적의 어려움**
   - "매일 네이버/구글 트렌드를 체크하기 힘들어요"
   
2. ❌ **수동 발행의 번거로움**
   - "같은 글을 3개 블로그에 복붙하는데 30분 걸려요"
   
3. ❌ **확장성 한계**
   - "블로그를 더 늘리고 싶지만 관리가 너무 힘들어요"

**해결책 (Solutions)**
1. ✅ **자동 키워드 수집**
   - "1시간마다 인기 키워드 자동 수집 + 카테고리 필터링"
   
2. ✅ **AI 자동 발행**
   - "키워드 → AI 생성 → 3개 블로그 동시 발행, 5분 완성"
   
3. ✅ **무제한 확장**
   - "언리미티드 플랜으로 매일 수십 건 자동 발행"

## 📐 Section 3: How It Works (작동 방식)

### 목적
- 3단계로 간단한 사용법 전달
- 비전문가도 쉽게 이해할 수 있도록

### 레이아웃
```
┌──────────────────────────────────────────────────────┐
│             "3단계로 시작하는 자동 발행"              │
│                                                      │
│   [1] ────▶ [2] ────▶ [3]                           │
│  연동       설정       자동화                         │
│                                                      │
│  ┌─────┐   ┌─────┐   ┌─────┐                        │
│  │ 🔗  │   │ ⚙️  │   │ 🚀  │                        │
│  └─────┘   └─────┘   └─────┘                        │
│                                                      │
│  블로그      AI API     매일 오전                     │
│  OAuth       키 등록    9시 자동                      │
│  연동                    발행                         │
└──────────────────────────────────────────────────────┘
```

### 콘텐츠

**Step 1: 블로그 연동 (1분)**
- 아이콘: 🔗
- 설명: "티스토리, 구글 블로그, 워드프레스 OAuth로 간편 연동"
- 시각 자료: 연동 화면 스크린샷

**Step 2: AI 설정 (2분)**
- 아이콘: ⚙️
- 설명: "본인 소유 AI API 키 등록 + 카테고리 선택 (건강, IT 등)"
- 시각 자료: 설정 화면 스크린샷

**Step 3: 자동 발행 (0분)**
- 아이콘: 🚀
- 설명: "예약 시간 설정 후 완전 방치! 매일 자동으로 발행됩니다"
- 시각 자료: 대시보드에서 발행 완료 알림

**무료 체험**

- 기간: 무제한
- CTA: "지금 시작하기" (Primary)

- 가격: ₩19,000/월 (1개 블로그)
- 발행 건수: 월 50건
- 블로그 수: 1~3개 선택
- 기능: 모든 기능 포함
- CTA: "선택하기" (Primary)
- 뱃지: "💎 BEST"

**스탠다드**
- 가격: ₩39,000/월 (1개 블로그)
- 발행 건수: 월 200건
- 블로그 수: 1~3개 선택
- 기능: 모든 기능 포함
- CTA: "선택하기" (Secondary)

**프로/언리미티드**
- 가격: ₩69,000~/월

- CTA: "선택하기" (Secondary)

### FAQ 콘텐츠

**Q1: 무료 체험은 어떻게 사용하나요?**
- A: 회원가입 후 즉시 5건 발행을 무료로 사용할 수 있습니다. 시간 제한은 없으며, 신용카드 등록도 불필요합니다.

**Q2: AI API 키는 어디서 발급받나요?**
- A: OpenAI, Claude, Gemini, Grok 각 플랫폼에서 직접 발급받으셔야 합니다. [발급 가이드 보기] 링크를 참고하세요.

**Q3: AI 생성 콘텐츠로 애드센스 승인 가능한가요?**
- A: 네, 모라브는 1500자 이상, SEO 최적화된 콘텐츠를 생성하도록 설계되었습니다. 다만 최종 승인은 구글 정책에 따라 다를 수 있습니다.

**Q4: 네이버 블로그는 지원하나요?**
- A: 현재는 티스토리, 구글 블로그, 워드프레스만 지원합니다. 네이버 블로그는 향후 업데이트 예정입니다.

**Q5: 환불 정책은 어떻게 되나요?**
- A: 구독 후 7일 이내 100% 환불 가능합니다. 발행 건수를 사용하지 않은 경우에 한합니다.

**Q6: 블로그를 나중에 추가할 수 있나요?**
- A: 네, 언제든지 대시보드에서 블로그를 추가하거나 제거할 수 있습니다. 단, 플랜별 최대 블로그 수 제한이 있습니다.

**Q7: 발행 시간을 자유롭게 설정할 수 있나요?**
- A: 네, 원하는 시간대(예: 매일 오전 9시)와 요일(월~일 선택)을 자유롭게 설정할 수 있습니다.
